# Web Scraper - Últimas Noticias La Nación

Este proyecto es un scraper en Python que extrae las últimas noticias del diario La Nación, procesa la información y la guarda en formato CSV/JSON. El objetivo es mostrar habilidades de requests HTTP, parsing de HTML con BeautifulSoup, manejo básico de errores y buenas prácticas de scraping responsable.

## Tecnologías

- Python
- requests
- BeautifulSoup4
- lxml
- matplotlib (para un gráfico simple en versiones posteriores)

## Estado

Versión 0.1:
- Extrae noticias de la página principal de "Últimas noticias".
- Obtiene título, fecha/hora y URL.
- Muestra los resultados por consola.
